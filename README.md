# The Little Taco Shop

The official website for **The Little Taco Shop (LTS)**, a fictional eatery serving the local community with fresh tacos. This project demonstrates the use of semantic HTML5, CSS layout techniques, and multi-page site architecture.

## 🌮 Project Features

* **Multi-Page Navigation:** Includes a home page, a dedicated store hours page, and a contact page.
* **Semantic Structure:** Utilizes modern HTML5 elements such as `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<aside>` for improved accessibility and SEO.
* **Interactive Elements:** Features a taco trivia section using the `<details>` and `<summary>` tags.
* **Comprehensive Menu:** A structured table detailing pricing for "Crunchy" and "Soft" taco options.
* **Functional Form:** A contact form designed to capture user names, emails, and messages, configured to post data to an external endpoint.
* **Store Information:** Clear listing of operating hours for every day of the week.

## 🛠️ Technical Details

* **Typography & Layout:** The site uses a base font size of 22px and a centered body layout with a maximum width of 800px.
* **Table Styling:** Data tables utilize 'Courier New' typography with collapsed borders for a clean, technical look.
* **Responsive Design:** Includes viewport meta tags to ensure the site is accessible on various device sizes.

## 📁 File Structure

* `index.html`: The landing page featuring the "About" section and the full menu.
* `hours.html`: Contains the weekly schedule for the shop.
* `contact.html`: Features the location details in Kansas City, MO, and the customer contact form.
* `styles/style.css`: The global stylesheet managing the visual presentation of the shop.
